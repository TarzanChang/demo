問題提問

請問老師,想請問若固定ip不是直接接數據機,而是透過hub出去的,可以設定固ip連外網嗎?
目前測試結果, 只要透過hub出去設定固ip就無法上網,只能直接接數據機設固i才可以,

老師回覆

對外網路的IP只能用，ISP提供的IP才能使用，所以你電腦直接對外設定的IP，不是ISP所提供的那就會不能上網。設定固定IP連線出去的方式可以詢問ISP，因為各家ISP提供的固定IP連線方式會有差異
